## 21.7.3~7.9
> <font face="Noto Serif SC">&emsp;&emsp;3号，今天等言的一天。据说是玩一个职业规划的沙盘游戏，好像很有意思的样子，“命运来敲门”，五种职业好像很是带劲啊。早上从实验室醒来，cyq突然出现来个实验室大搬家，就一起加入了战斗。这么从实验室搬到楼下，再从实验室坐货拉拉一路到电信住宿大楼，这个小房间怎么说，虽然小，但毕竟是单人间，再跟靖靖坐车回去，至此估计cyq是真的毕业咯。回去洗了个澡，准备出门，这瓢泼大雨就下来了，这应该是几年来下雨骑车遇到过最大的一次了，打伞都没什么用。晚上虽然分开了两天，在地铁站偷偷摸摸地等她，见到她的时候就有一种好久没见想拥抱她的感觉。这大雨那我们就去到好地方那必须就是出地铁就直接能吃饭的地方，而且不用沾到一滴雨的地方，没有错，那就是苏州中心。我们聊到了一个话题，关于一个女生说自己想通过自己的努力提高自己所在的阶层认识更好的人然后再谈恋爱，被老师认为是错误的，因为三十几岁的男人只会喜欢二十几岁的女人，彼时的她已经不具备竞争力，她认为这个观点很好，我也就觉得反驳得不错。然后她就开始了灵魂拷问，让我选择在三十岁的时候一个三十几岁很优秀的女性和二十几岁的青春姑娘，这就很夸张，我说我没体验过三十岁不清楚，拗不过我又说这个问题一旦被问出来其实答案就很明显，就像套数学公式般，没有具体的例子，没有选择人具体的前提条件，那么答案就只能是后者，因为如果量化成数据加上公式答案就显而易见，但是人类这种生物本身神奇的地方和爱这种的事物神奇的地方就在于，它是不可量化的，所以这是一个很差的问题，没有存在的必要，泪目。晃着晃着到了南京大排挡，聊到了设计一个算法组合所有光大网点的半径一公里的房源，进行最优查找，我简直是个天才，或许真的可以实践一轮，这个大排档除了评弹好听一点好像真的就没什么优点了。又get到了苏大雨神的外号就很刺激。<br><br>4号，等烘干机一直等到了中午，我们才到了理工楼。今天周末跟言玩的很开心，顺便分享了一下我的指静脉小制造。晚上回来突然发现宿舍变成了三人间，瞬间有些伤感，这属实没有想到会发生这样的事情。晚上回来，跟鸿雨聊了聊工作，喂猫与猫粮，他想明年带走几只猫，我们都想在苏州定居，去北京就是赚钱，没其他的，他想现阶段就想摸鱼到毕业，不过他确实倒是没什么忧患的感觉。今天晚安的奇妙之处就在于居然加上了我们都显而易见的称呼，挺神奇的，但也挺奇妙的，就好像我知道这是我的宝贝娃娃，睡觉的时候对它说：晚安，我的宝贝娃娃。<br><br>5号，今天农行来了通知，言：你的自由要被剥夺了。<br><br>我：不，过度的自由何尝不是一种禁锢呢，自由不能以牺牲自由为代价，什么是好的东西，大多数情况下是好的并且总能带来好的结果，什么是good，有的人认为钱多多是good，但是当good多了之后就变成了goods，我们都知道goods是商品 它是为了盈利而存在，此时此刻它已经变味了，泪目。<br><br>言：这一通绕我差点没明白，不过goods怎么就是为盈利而存在的啦，商品本质上是一种劳动产品，是人类生活所需要和享受的东西，只是现代社会赋予了它交换和盈利的属性，就比如你家有一座山的苹果树，你就偏不卖，诶，就是玩。<br><br>我：这个只是历史学的角度，咱们只讨论狭义的goods(你这可是在反驳罗老师)，你还别说，我家就是有棵栀子花树，有人出价，哎，就是玩。<br><br>今天还是码字的一天。<br><br>言问：你在干嘛<br><br>我：利用移动性指针控制仪器，以及 电子机械键块上下控制装置，和 一块像素矩阵组装的不断刷新的发光物体，进行 文字创造过程以及信息浏览工作。<br><br>言：懂了，玩电脑。<br><br>我：....悲伤的故事，不要总结得这么泪目，真的是像个男人的聊天方式，总结完了，这天还怎么聊呢。<br><br>言说她要我帮忙收快递，我写到这里时已经过去一周了，也没有音信，不是很懂。今晚她已经表示光大的饭不好吃了，跑去观前吃了，事后证明这是个错误的选择。我们知道了switch开门了，我驻足过很久的那家大酱店关门了，泪目。然后就开始下起了大暴雨，两个人要是从观前走回去，那就更惨了，不过坐完地铁从升降梯出来，淋了个爽也很惨。今晚的峡谷之旅也很炫酷，录下了精彩瞬间，我跟言说队友会说我的五杀你抢了四个，她后来才发现这句话的笑点，绝绝子。深夜收到她的告状，南京大排挡的桌子把她的膝盖撞青了，我表示拉黑这家店，这个设计确实很绝，桌子下面设置了一个平台，腿没法伸不说，场地高度不一样，下面的食客的头对着上面食客的腿，无敌。我只说青青的膝盖不能被人发现引起不必要的误会，她说为啥子呢，一时之间我竟不知道她是真不知或又如何呢。不过她也认为扬州是最好吃的地方，还是很不错的。<br><br>6号，这是早上5点半不到就起床的一天。言说我很不尊重芋泥，好，我走，泪目，居然不给亲亲，让我去找奶茶亲亲，关键我没点奶茶。今天亲戚来访，没办法我也帮不上忙，只能念叨它干净走了，遥想当初我们总能在这个精准的时刻做一些特殊的事情，我被踹了还被盗图，这上哪说理去。这一天ACG战队关于如何从5人变成4人的故事，曾经被踢的那位仁兄曾几何时在我面前质问我怎么踢的他，我只表示他居然还知道自己加过这个群还知道自己被T了。袁哔哔建议我下线省点电，我建议他少敲几下键盘省点命，乌雕云建议他早点自尽省点地球上的资源，我建议他赶紧自尽前销毁自己的数据以及纸质资料更加节省地球资源。今晚唯一意外的项目是宿舍聚餐，众所周知，我们宿舍我只有一个比较熟的好友，平时四个人白天都不在宿舍，晚上我回去大家又都睡了，所以即使两年过去了其实也没什么交流。主要项目就是聊聊大家将来的去向，和主要是NLP实验室的趣闻，毕竟我实验室在整个院不出名也没有大名气的老师，他们正好有两个是NLP实验室的。史称「苏城聊科院师趣闻事件」，以及《计科老师那点不可描述之三两事》。哦对了，这一天还是接吻日，不过鉴于我们没有哪一天不是，就不必在意了，不过在这样反倒是这样的日子，我们反而没了机会，就挺泪目的。回来感觉好累，就去跑了个步，洗了个澡，刚看到言5分钟前的消息，她就已经进入了深度睡眠。打了个电话试探了一下，挂掉了，但是她又打了回来。本来计划要玩一个侦探游戏，这下我脑子乱了，一股脑分享了一下最近奇怪的事情，诸如宿舍走廊拐角处的仰卧起坐椅上突然出现的日本武士刀，过了几天又没了，言一开始看图片还没看出来，之后才毛骨悚然起来，还有小治在地铁上接收到来自隔空投送的半个月前的图片，以为是超时空投递的一张艾斯奥特曼加上文字“站起来，让我坐会”等等。最后只能意犹未尽，各自睡去。<br><br>7号，言说昨晚舍友睡得太早，所以自己就也睡着了，然后我打电话就没接着，其实我打电话就也是试探一下，没想真打，但她打回来了，但其实此时她舍友睡着了，本来准备好多今晚聊的天，我突然摸不着头绪了。然后她上午跟我说完就好久么有回复，我以为她在迁怒，实际是在上课，后来她才告诉我经过上次事件之后她不再随意生我的气了，这我就明白了。她居然忘记了顾飞买给冬娜的她想吃的中式汉堡，那我只能一个人独享了。上火已经快好了，才告诉她，并且这个时候我才决定去买点西瓜霜虽然已经意义不大，但可以为后来的上火作为铺垫吧。骑着车从东区西门出来直接左拐一路直行，这让人想原地火化的太阳。我以为的医保买药： 刷医保卡价格改了，然后用支付宝支付，实际的医保买药：刷医保卡里的钱买药。泪目，以前怎么没听过这个事，啪啪哲说是压根没有学生会犯我这种错误，很强，居然没有愣头青跟我犯一样的错误。今天一大清早ACG战队就吵了起来，袁逼逼表示打团AD永远不在，乌雕云表示人家优势故意卖破绽让你打你还就真上了，泪目，悲伤的故事，然后开始互相敷衍，假装认错现场。今天还是大型起名现场，袁里克(沃里克，约里克)，吴法天(无吊所谓)，叶索，叶青(李青，亚索合体)，吴克莎(金克斯，卡莎合体)。迪迦！盖亚！！戴拿！！！我才知道盖亚戴拿的变身器叫蓝宝锥，闪光剑，泪目。估计是第一次组会在实验室开的一天，介绍了一下我关于指静脉的成果，各人介绍自己的工作，宝俊一马当先近期成果主要是关于tf联邦学习的搭建，旺旺介绍了不老少论文，浩浩也慢慢赶了上来。今晚又是蓝屏侠乌雕云一枝独秀的一天。今天视了个频，很开心。<br><br>8号，今天终于等到言的拍一拍：爷爷别生气，爷爷我都等了多久了，泪目。今天又是大雷雨，天的乌蒙渗透入了荫绿，这种绿蒙的状态令在场之人(只有一个)无不想起当年那部《The Mist》以及见到最后那只庞然大物的绝望感。那处的光芒仿佛要天神震愤，一片光，仅是一片光，与我们的绿区分了开来，但在大雨中便渐融为了一体，不再有隔阂。值得注意的是，言喝完班长请客的三兄弟，问我猜她在干嘛，至今都不知道答案。今晚又是蓝屏侠乌雕云一枝独秀的一天。今晚视了个频，很开心。<br><br>9号，听闻言的科技岗舍友马上要离开，悲伤的故事。今天主要是了解了一下苏州农行和邮储软开等各个银行科技岗位的比较，农行不错，邮储工资高但要借调北京，其他交行不行，建行不错等等，农行试用期一年，第二年24w。下午盘算了半天跟言去哪里吃，她一句晚上老师在不能走，那只能作罢，转手打开饿了么，毕竟靖靖刚刚走，这时候去太尴尬。北京烤鸭不错，odd taxi看着也很爽，但是欢乐时光总是如此短暂。我们的夜晚逐渐缩短到一定境界了，而我也无法前往她的所在地，因为害怕目光。</font>
