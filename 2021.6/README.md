萨缪尔约翰逊说伟大的工作是靠耐性而不是力量去做。读书的四种境界：在书籍中逃避现实。长恨此生非我有，何时忘却蝇营。夜阑风静。小舟从此逝，江海寄余生。没有摆脱逃避的本质。在书籍中营造世界。在书籍中完成自我救赎 SC">&emsp;&emsp;23号，码字。这一天，我直到中午11点才收到她消息，以为她手机静音了，总之我打的电话都接不到。情急之下，想让学妹喊她起床，可惜学妹上上课，所以只能作罢，后来她回忆说早上起来手机都找了半天，手机对话框还写着：可能是熊校长提到的。总之生气不生气的，一会会儿就好了。大家似乎忘记了今天组会暂停了，直到我中午提起，才长吁一口。晚上，收到了ycc同学在长沙带来的茶颜悦色的盲盒，期待了好一会儿，打开一看，我俩是一样的，害。下午录制完了整场毕业典礼的演出，自己先回顾了一下，尤其是为李麟青博士授予完学位后，全场灯灭，那一句深远悠长的：长亭外...响起。晚上又带言回顾了一下，我能感觉到，听到那一声的一刻，一声抽泣。今晚意外地，我很早就开始困了，神奇的故事。<br><br>24号，今天又是个特别的日子，总之就是浪漫的夜晚。品尝了一家叫谷川的日式西餐，居然已经开了14年了，还真的是没听过。看《遗物整理师》到泪目，她倒是没感觉，但是她晚上看《生活家》泪目了，我倒是没感觉，有点神奇。<br><br>25号，几乎是中午才出发，顶着大太阳放弃了降龙爪爪，尝试一轮囍先生茶餐厅。明明是中午，而且还是周五，却是异常地人多，坐的位置的墙壁给我一种在澡堂吃饭的感觉。不出所料，我们又点多了，但是还是很棒的，尤其是那个超级销魂饭，有当初那种在港丽的感觉。那个电风扇，让我又想回顾一遍《国产凌凌漆》，她不喜欢星爷，那只能我自己独自回忆了。<br><br>26号，今天是打球的一天，没抢到场子，只能苟到一个小场子，打的一点都不爽，被一堆老师占场。为了缓解视力疲劳，看着校园绿色风景，又逛到了南门，不知怎么的又到了双塔。终于看到双塔的菜市场属性，尝了一尝豆腐花和我依旧最爱的赤豆糊小圆子。<br><br>27号，这天下午是长久以来，一起玩游戏最爽的一次了，殊不知痛苦的事情永远紧随其后。发生了一些悲伤的事情， 说起来还是聚会这档子事。这是研会的最后一次聚会，想着只有她一个人参加，这次一起参加弥补一下缺憾，但今时不同往日，人已经变了。已阅不回不是新鲜事，当演员我也没理解就过去了，后来才知道她很不喜欢他们开我俩的玩笑。导致我全程想发个信号很难接受得到，她都不怎么看我，最后结束了我是真的很想跟她一块回去，然后她说我们回东区你自己去玩吧，这边说给人家一点自己的空间。我没法只能去唱歌，这么多人唱不了几首歌，然后我耳朵还吵死了，越想越不对劲，感觉今晚我俩像个陌生人。回去想当面聊一聊，可惜没有机会，只能在电话里面聊，我就想当时一起回去就没这么多事了，还说我是碰瓷。她说完全忘记没回这档子事暂且不提，我是真的没感觉到他们开玩笑的冒犯，但确实是别人的过错，不应该我们自己在这生闷气。总结经验教训还是交流最重要，如果我们一开始能聊通今晚怎么过，或者后来能回去聊，就没这么多事情了。<br><br>28号，昨晚做了一个噩梦，梦见我看不见她的脸庞。下午又聊了好一会儿昨天的事情，误会还是完全没有消解。码字。今晚开始了第一次的《轮到你了》的解说观看，我就知道她会喜欢看，一晚上看了十集左右，根本停不下来。<br><br>29号，“我会早起的”，今天以这句话开始我觉得非常合适，总之就是我将近十一点的时候才看到她，之后就和ycc神奇般地一起出发。她去跟高中同学聚餐了，很神奇，在苏州能聚到一堆南通的同学，这顿饭据她的话讲，吃了两个小时的翘脚牛肉，两个小时的奶茶。我开始了我的MATLAB指静脉代码转PY的工作。这天晚上又看了好久的《轮到你了》，直到剩下最后两集，留到明天看，这个夜晚也是最后的夜晚了。晚上翻来覆去睡不着，想着这持续了九个月的日子，已经要一去不复返了，再也不用晚饭的时候来到女生宿舍楼下等她半天回去吃饭了。再也没有晚上去操场跳绳了，虽然我们总共也没跳几次。再也没有回到宿舍觉得她只在隔着一条马路的隔壁楼的里面。也许这些再也没有，有一些也不是没有机会再体验，但是我的夜晚也终究要改变了，这一刻来得如此之快，这一晚我只觉痛苦万分，我知道她是无感的，到了明天晚上，她才真正感觉到。她睡得太晚了，以至于今晚罕见的是，我先睡着了。<br><br>30号，上午继续写指静脉的代码，中午跟言吃了近期在学校的最后一顿饭。已经很久没有中午一起吃饭了，也很久没吃过东区的饭了，记得当初言说辞职后白天跟我一起去实验室怎么度过。事实上，辞职之后就直接去旅游了，回来后也每天都有事，几乎就没来过，泪目。这一顿吃了四食，看到了带闪现的ycc，一起又逛了次超市，坐下的时候看到桌子上的拍立得照片，男生长得像猴儿，我们又聊起了卢本伟的光荣事迹。马不停蹄地赶回宿舍，真的是，第二次进入女生宿舍，一如我想象中的模样，有点类似，又有点不同。两个人搬到力竭，顶着大太阳，推着小推车去西门，言还不忘回头来拍我。搬着十个包裹到了酒店，恐怕有这么多行李的也只有她了趴，泪目。终于看到了《轮到你了》大结局，故作玄虚，我估计此时她心情跟我当年一样，这什么阑尾结局，怎么看都是要捧一波西野七濑的节奏。看着看着，舍友拖着包包来了，欢乐时光总是如此短暂。后来就逛到了观前开始神游，喝到了心心念念的一点点，闲暇之余，看了一场之前了解的《你好世界》(Hello World)，这片岂不是程序员最爱？！看开头的时候，真觉得这么无聊的片，还不如看番，昏昏欲睡，除了一点出奇的设定就是，男主本身是数据，被外部的未来十年后的真实世界的自己告知这样的事实，然后要帮助自己追到女朋友，完全没有剧情冲撞点，没什么内核。真正开始转折变得精彩的地方应该是男主成功利用手套的力量抵御住了雷击，阻止了本应被雷击中的女主的死亡，然后未来的自己开始了真正的计划，将女主带走，男主才知道自己被骗了，将她的思想灌注到现实十年后已经脑死亡的女主身上，这样她的脑部复苏后的记忆就跟十年前一模一样，不得不说很精妙，此时不知道心疼生活在数据世界的男主还是心疼十年后的他，两个人都是同样的痛苦，一个因为刚爱上了一个人就被带走，一个爱了十年的她通过各种研究物理量子，实验了无数次只为了复活自己所爱。但是数据世界中的男主失去他所爱之人，也是那样令人泪目，当然传奇的地方也在于，真实世界受到了模拟世界数据的影响，最后男主获得了自己的所爱，而为爱痛苦了十年的他也死在了与系统的战斗中，尤其是他对即将离开的十年前的虚拟世界中女主时说的：「不，我不是坚实，我只是个陌生人」的时候，真的是无法言状这种感觉，明明上一秒还是如此地放不下她，坚持了整整十年，突破了科学与自己身体极限只为见到她，可此时只有放下的痛苦...最后还是觉得是一部值得推荐的电影，看到现在确实对于我来说还没有哪一部动漫电影不好看的。回学校的时候，下地铁的瞬间，她想到此时此刻的校园是如此的遥远，她才发现自己已经毕业了，校园已然没有自己的容身之所，我看着她的眼泪打转，倔强地不想流泪。在实验室度过短暂的时光后，喝完※霸客，终究是要离开的，我知道就这样让她一个人回去，未免太孤独了，我一定要送到最后，还是下地铁，这一次哭得更厉害了，我能体会那种感觉，提到分别没有感觉，真正分开的时刻才是鼻子最酸的时刻。我说没事的，这么近，我也在，大家也在，想回来也很方便，相见也很方便。我知道过不了几天她就会忘记这份悲伤，甚至跟新同事在一起都忘记了我的存在，但是此时此刻的伤心是真实的。终究我还是要回到这一方土地，继续我的修行，还有很多问题，很多人生需要去探寻，这一个月的时间我们还有很多事要做。<br><br>1号，今天是言真正离开学校的第一天。大清早就很生气，老是忽略我的消息，总之就是一起远程同看党的100周年庆典。尤其是听到那两句“万岁”，当场泪目。几乎又是写了一天的指静脉的代码。<br><br>2号，今天是分开第二天了，越来越像过去，但是关于签到的事还是很在意。早上还是没起得来，搞了一天的指静脉识别软件，基本上是搞出来初版，但是离真正的应用还是很遥远。</font>
