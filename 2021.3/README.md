3.11~14，11号跟言一起学习了一天，我鼓捣电脑修复了一天，言改了一天的论文。12号，言去预答辩，结果还不错。晚上，宝骏和靖靖给带来了一些水果，这是他们唯一能关心我的方式了，很暖。此外，又多了些美妙的回忆。13号，在和府捞面共进午餐，我们聊了岛国特色科普，在星海领到了被我抛弃了七年的医保卡，晚上回到家，与母亲一起度过了夜晚。14号，与言第一次玩了GTA5和Florence，那个凄美的故事，预示着夜晚。一起又吃了双塔，我说你好像总是没有烦恼呢，当然只是感慨。走着走着，终究聊到了一些我一直揪心的话题。起因是言说七月父亲大人来，肯定要见我一面，免不了很多聊天。我总是想找个机会，此刻的轻描淡写，积聚了多少心理准备。一个人的烦恼好像突然变成了两个人的了，当然其实之前我也说过了。我知道这与父亲大人无关，这终究是我的问题，我希望我有能力解决，可是时间会给我机会嘛。

3.15~17，有些失魂落魄，三天针灸结束了，好像还是没什么反应呢。一个病友说她得母亲一周看的，十针好了，她刚出问题就来了，扎完三针也效果很好，不知道我两天后会怎样呢，畅玩了啪兄刚租的司娃渠，品了一品啪兄滇红大芽，确实不错，七八壶下去还是很红，有一种红薯的红茶味，从各种茶具聊到手上的串，品完回去就有种强烈的产品赋能的冲动。
