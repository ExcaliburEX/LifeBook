light">&emsp;&emsp;23日，⼤概是同等学⼒监考⼀天，跟护理学院的同学聊了她所经历的汶川地震与护⼠⽣活。<br><br>24日，虽然闹了点小脾气，但看到她的脸就生不了这个气了，就像她当初看到我就狠不下心一样。今天是跟她和她同门一起切磋球技的一天，就很带劲，记录一下第一次一起打羽球~🏸。<br><br>26号，昨天艳阳高照的大晴天🌞，今天开始从早到晚的雨☔。上午送言去做光大的体检，在市立医院，关于是否能快速拿到体检报告单的一波大讨论。中午送言去东区，又被安利了一波东区的炒酸奶，自己舔起了冰淇淋，真的是很乖巧地在舔，这年头已经不多了。下午我继续修缮我的组会报告，也就是日常记录，顺便参加了年纪大会，就是书记介绍新来的辅导员以及等等思想教育，值得注意的是我们只剩下四个月写论文了。言整整在答辩开始后的一个多小时才给我回复，疯狂吐槽了一波罗奶奶的凶狠，只有她们这组这么压抑，其他组都很轻松。最后结果还是很美满的，顺利通过答辩，舍友也是美美地度过了答辩，另外pyq也有很多法学，商学的同学传来了顺利答辩的喜讯，这一刻他们好似已经从这里飞远了。夜晚🌃，我去参加组会，她去美美地与舍友干掉一顿牛蛙自助，美滋滋😋。晚上，她忘记跟我说晚安了，一夜睡不安稳。<br><br>27号，这一天在草坪那里坐了很久，聊工作的事情。回来待了一会儿就开始了第一次跳绳训练。我印象中我会跳绳啊，结果就是每次跳不过十个就停下了。但是还是很happy的，由于刚刚锻炼完而且看得到跳绳上记录的圈数与卡路里。到了超市便看一样商品她就开始计算每件商品的卡路里，换算了半天。化身为卡路里数学家的她，逛了N圈超市终于算明白了一杯柠檬茶的卡路里值，很强。<br><br>28号，8点半我开始打电话，听到对面软哝萌萌的声音的时候，我就知道了一切。等人的时候，忠太打过来咨询工作以及通过IP能否定位手机号的问题。她说是为了锻炼面试能力，在外面等了一上午的面试，艰难的体验。下午她去拿体检报告，宝骏过来说能否认识一个人。阴差阳错，还真的找到了，本以为可以通过舍友进行作战，但被言一票否决，虽然我也觉得有些不妥。晚上言签完约，跟舍友在观前吃完湖南常德面，我忘记吃饭了。深夜，就又一起来到了老地方，了解到了舍友作战失败的血淋淋的教训，结论就是多在pyq分享帅照，点餐也点到了印象城金拱门店。她克制不住地想给我修眉，别说还有点意思。<br><br>29号，中午她的谢师宴以及一下午的剧本杀。看了一下午的《Heaven's Feel Spring Song》宝骏和靖靖和旺旺和珊珊一起去独墅湖玩，审片，拍了很多照片。一起聊了一聊关于希腊神话中没说美杜莎会骑马那么fate heaven' feel中Rider美杜莎的宝具是「骑英的缰绳（Bellerophon）」操作天马进行冲转是否合理？就类似于没有史料记载李白是否会游泳，那我后面说李白会游泳是否合理，虽然说李白是淹死的，在水里捞月亮时，他已经清醒了，如果他会游泳应该淹不死才对，但是淹死的都是会游泳的，但是问题又来了，没人看见李白落水谁又知道当时的他会不会游泳呢？这就像我没没看到史记上说刘邦会不会跳舞，我是否可以说刘邦会跳舞呢？一起散步到了平江路，聊到了关于安禄山与杨贵妃的故事。跟言一起漫步平江路，聊起了她下午玩到泪目的《无名之辈》的故事，这是一个关于七个人在一起回到拆迁房时，拆迁房崩塌全员死亡因为怨念一起穿越到20年前一起生活的小院里解开多年间各人与其上一辈之间的误会矛盾的故事，众人读至此，无不潸然泪下，涕泗沾襟。这一晚走了很远的平江路，创造了以往的记录，但还是没有走到头，但是巧合的是在我们讲着白天玩的剧本杀时又偶遇了一个角色扮演的剧本杀馆「推理社」。回来看相机里面的照片，看完了今天拍的，顺便就把16年开始拍的照片全看完了，全是我的丑照，其实15年也有不过都删完了。我说那不行，你都看完我的了，是不是得看看你的了，这个就很刺激，一直看到了很晚。<br><br>30号，白天她边看剧边写材料，没写完。看了一下午的《Heaven's Feel Spring Song》吃完晚饭，聊了一聊最近的新型骗术与毒品。散了一散步，一起玩了一会单杠。欣赏了一会儿《生活家》，又是边看剧边写材料。<br><br>31号，悲伤的是没有完成论文目标，傍晚骑车的时候她说今晚继续跳绳吧。总之这么主动运动，那我们必然是要赴约的嘛。今天风很大，跳绳就非常爽，虽然还是不连贯不过我已经进步很大了，她有点被我带跑偏。结束后，去到超市，碰到了小治，出去超市小治自己走了，她老让我跟他走，自己一个人骑车，这是人话啊。我正想起来今天的八卦想跟她说，就发现宝马(她的叫法)他们俩正好出现在前方，这时小治刚走没一会儿。她就驱车歪歪扭扭地冲向人家的前方，以她那拙劣的车技在前面扭了好一会儿，事后问起来发现他们压根没发现我们，就很神奇。骑到停车地，我说他们肯定会先去女生宿舍，她比我还激动，说着走啊走啊，赶紧跟上，然后一个箭步冲了出去，我说你怎么比当事人还激动，我顺便还给小治发了信息，随手拍了张照片，我以为这是以后他们人生中宝贵的一张照片，想不到却是最后一张照片。宝马dd回头正好看见我们，聊了一两句，她把我推走，让我赶紧跟宝马回去听八卦，自己直接跑回了宿舍，我跟不上只得作罢回头，还是很秀的。回来三人在宝马dd的宿舍聊了很久两人这几天以及今晚的细节，想起当初我也是这样的夜晚，很紧张地过来聊天，然后又是这样的夜晚，我们都很兴奋且相信这局稳了。但仔细想想，既然又这么多相似的情节，但有些情节为什么没有复刻呢，难道这就是后续一系列事件发生的理由？原来是事件的顺序交换了。晚上等她的时候，在路灯下跟老妈通话，关于以后住不住一起的问题，我被骂得钻心，不过我暂时也不想再想这些事了，以后她总会想通的。<br><br>6.1号，关于儿童节的故事。<br><br>我没想到是以这样的方式开始，突然大家一起兴起，我就去买了很多零食。在众人的手机与目光的照耀下，我开始了零食书包的制作，众人绘声绘色一波强劲商业互夸，完成了我觉得非常酷炫的儿童节礼物。剩下只待她晚上出现，众人一起记录下这有趣的瞬间。但她晚上要整理论文材料，而且下午的时候我们才得知宝马dd的噩耗，众人皆不知如何是好，只得看他阴沉度日。一时间，我们都想起了两年前的事件，都是如此惊人地相似，难道这就是我们所不得不背负的达摩克里斯之剑？只消等待它落下的方位与时机。晚上，雨夜，很大，她很困估计且兴致不高，不过我一定要前来，我知道她一定会很喜欢。背着包第一次坐在里面的沙发上，我还是先跟她说了此次比较悲伤的事件，以及下午看到的女生呆坐于雨中的泪目情景，这个儿童节我看到的更多是悲伤。然后我故作神秘地从书包里面掏出了“书包”，她那个欢喜程度一下冲消了我们当前两人悲伤的氛围，抱着爱不释手，非要拍照，我说本来是下午等你来给个惊喜的，不过现在也挺不错的。这就是儿童节的故事了，真实且悠长。<br><br>2号，毕业照时刻！迫不及待想到现场看看毕业照盛况，顺便在旁观者的角度拍到穿自己衣服拍照的瞬间，后来发现这张照片效果意外地比正面拍的好，反而可以成为毕业照精品之列。结束之后就是宿舍四人加上两名男友的各种合影瞬间，从法学院到钟楼草坪再到东吴门，走过这些看过不到无数遍但也有些记忆痕迹的建筑。发挥了我的特写优势以及捕捉到了拍立得之美，学妹来了之后第一句：你们还请了摄影师？不错，我这个今天账应该能结到了。有些瞬间被永久镌刻下来，记忆的音符在跳动，虽有瑕疵，但总有弥补的瞬间，我们还有更多的机会。晚上组会结束之后，一行人一起跟宝马dd去吃了龙虾，希望能一起安慰的同时能有一些出谋划策，依稀想想以往每次在一起吃小龙虾都是我的悲伤时刻。回过头来，还是赶场回去跟她一起散步。<br><br>3号，这天又是一如既往的雨天，她没带伞，直接去了东门，转而一路飞驰到了仓街的真实的泥泞崎岖之路去赴她舍友的小龙虾之约。又没法一起吃饭，只能每一刻多想在一起多待一会儿了，舍友一直没来，我就知道我幸好没走，一个人等待小龙虾是多么痛苦的旅程。等她唱完KTV，已是深夜只能等待她回去洗澡，然后再出来相会。洗完澡已是深夜，但是今晚不见就睡不着，也不能浪费长时间的等待不是，依然是冒雨的相见。应该是最晚的相见时间了，12点半？<br><br>4号，这几天虽然我们还是能每天都能见到，但我还是感觉见她一面太难了。今晚必须打卡之前的心愿，谁知道她明天又在何方呢，果不其然，后来的事情验证了猜测。她想了半天没想出来我们之前说的那个东西，只能说这段时间发生的事情太多了。椰子鸡还是不错的，汤很香，鸡很美，周总的MV专场，一起回忆了那些年的古老歌曲。从观前走到东升里，想着以后拍摄怎样的视频，终于在这样的夜晚回归了以前的routine：一起看视频。先补完了暴走，再继续补木鱼水心，就这样安静地摇着摇着，慢慢地睡着了，算是近期回去比较晚的一次了。<br><br>5号，这一天，她回去给外婆过生日了。选了半天的头戴式耳机，决定换一个JBL。傍晚的时候跟老于一起四个人一如既往地聊了一下各种工作的情况，基本现在的选择除了三大运营商和银行，也没有其他的实习了，但是去了这些地方后续也没法去互联网企业了。撸得兴起，还当众翻了两个空心跟头。</font>
